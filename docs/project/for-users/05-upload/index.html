Uploading bots involves getting your bot design as a VoiceFlow .vf file and uploading it in our app.

Getting the Voiceflow File

First, go and download your VoiceFlow design as a .vf file. Instructions HERE. You will likely need to be on their paid plan to do that.

Uploading

You can upload that file at the upload screen, after which you can add a name.

Attention! Two bots cannot have the same name, so make sure the name you choose is new and unique.
