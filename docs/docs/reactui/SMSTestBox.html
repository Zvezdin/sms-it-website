Class: SMSTestBox

SMSTestBox()

Displays a smartphone with a text area and send button for input. Allows the user to interact and have a coversation with a simulated version of the selected bot. User input appears on the right side of the smartphone, with bot replies appearing on the left. Used for chatbot testing purposes.
