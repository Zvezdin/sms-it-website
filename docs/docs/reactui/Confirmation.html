Class: Confirmation

Confirmation(intent, message, accept, cancel)

Displays a confirmation prompt over the current page. The page is darkened with a transparent overlay to put focus on the prompt.

Constructor

new Confirmation(intent, message, accept, cancel)

Parameters:
Name Type Description
intent INTENTION

the action that needs confirmation (enum value: 1 UNLINK/2 DELETE).
message string

the message to display on the prompt.
accept function

reference to the function to call when 'Yes' is clicked.
cancel function

reference to the function to call when 'Cancel' is clicked.
